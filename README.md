# Movie Recommender System

Système de recommandation de films utilisant le machine learning.

## Objectif
Ce projet a pour but de recommander des films similaires à un film donné, en utilisant une approche de similarité cosinus sur les caractéristiques textuelles (synopsis, genres, acteurs...).

## Technologies utilisées
- Python
- Pandas / NumPy
- Scikit-learn
- Streamlit
- Git/GitHub
